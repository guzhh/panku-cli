# 前端脚手架文档

## 如何以前端脚手架创建前端项目

### 盘古 前端 cli 工具

1. 安装

   ```sh
   npm install -g panku-cli
   ```

2. 命令介绍

   1. 查看可支持命令 `pk help`
   2. 查看版本  `pk -V`
   3. 创建项目 `pk init 项目名`
